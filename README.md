# 🚀 Transparent Public Complaint Management System

A full-stack web application designed to improve transparency, accountability, and efficiency in handling public complaints such as road damage, water leakage, sanitation issues, electricity problems, and more.

The platform allows users to raise complaints, admins to assign them to departments, and department officials to verify and resolve issues while maintaining complete workflow transparency.

---

# 🌟 Features

## 👤 User Module

* User authentication & authorization
* Raise complaints with:

  * Title
  * Description
  * Images
  * Location
  * Category
* Track complaint status in real-time
* View complaint history
* Mobile responsive UI

---

## 🛡️ Admin Module

* Secure admin dashboard
* Manage all complaints
* Assign complaints to departments
* View analytics and insights
* Track complaint progress
* Monitor SLA & escalations
* Role-based access control (RBAC)

---

## 🏢 Department Module

* Department-specific dashboard
* View assigned complaints only
* Verify complaints on-site
* Update status:

  * Pending
  * In Progress
  * Resolved
  * Rejected
* Upload verification proof
* Location-based complaint handling

---

# 📊 Analytics Dashboard

* Interactive charts and dashboards

# 🔄 Workflow

```text
User raises complaint
        ↓
Admin reviews & assigns department
        ↓
Department verifies issue
        ↓
Status updated (Resolved/In Progress/Rejected)
        ↓
System tracks complete transparency
```

# 📂 Installation

## Clone the repository

```bash
git clone <your-repo-link>
```

## Install dependencies

```bash
npm install
```

## Run frontend

```bash
npm run dev
```

## Run backend

```bash
npm start
```
